# Changelog

本文件用于记录每个版本的对外变更。

建议规则：

1. `v1.0.1` 这类版本用于小修复
2. `v1.1.0` 这类版本用于新增功能但不破坏现有使用方式
3. `v2.0.0` 这类版本用于重大结构调整或不兼容变更

建议每个版本都按下面 4 类整理：

- 新增
- 优化
- 修复
- 注意事项

---

## v1.0.0

发布日期：2026-04-24

### 新增

- 支持 `C4D` 手动渲染完成通知
- 支持 `C4D` 渲染队列完成通知
- 支持超时提醒
- 支持飞书 `Webhook`
- 支持 `Server酱`
- 支持通用 `Webhook`
- 支持托盘常驻
- 支持开机自启
- 支持静默启动，无黑框
- 支持浏览器 Web 控制台
- 支持中文界面
- 支持通知模板配置
- 支持通知字段编排

### 优化

- 将最终控制端统一为 Web 控制台方案
- 优化托盘交互和状态展示
- 优化发布目录和交付文档结构

### 修复

- 修复 watcher 与控制台多入口带来的启动混乱
- 修复通知字段保存异常
- 修复通知模板回退默认值异常
- 修复页面缓存和旧后台进程导致的前后端不同步问题
- 修复日志与历史信息渲染异常

### 注意事项

- 当前配置默认存放于 `%APPDATA%\TongzhiRenderNotifier\`
- 如果迁移到新机器，请一并复制 `tongzhi_render_notifier.json`
